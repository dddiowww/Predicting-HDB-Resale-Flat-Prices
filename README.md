# Predicting HDB Resale Flat Prices

This repository contains the course project for **CS5228 – Knowledge Discovery and Data Mining**.
The objective of this project is to **predict HDB resale flat prices** in Singapore using various data sources and machine learning techniques.

---
## File Descriptions

### `feature.ipynb`

This notebook focuses on **feature engineering**, specifically:

* Integrating auxiliary location data into the main dataset
* Generating and merging new spatial features with the training data

### `script.ipynb`

This notebook covers the **data processing and modeling pipeline**:

* Feature engineering
* Training and evaluating advanced models
* Saving the predictions for submission and analysis

### `history/`

This folder contains previous versions and experiment scripts, which document the project’s progress and earlier approaches.
These serve as references for methodology comparison and result tracking.

---

## Objective

The main goal of this project is to analyze key factors influencing HDB resale prices and build predictive models capable of estimating resale prices accurately based on available features.
